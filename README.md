# GOLD CLEAN

A modern bilingual corporate portfolio developed for **GOLD CLEAN**, an international manufacturer of professional cleaning and hygiene products.

The website presents the company's identity, products, certifications, manufacturing capabilities, logistics, target markets, and strategic vision through a clean and responsive user interface.

# Installation

Clone the repository

```bash
git clone https://github.com/jalalmahmoud8000-debug/gcport.git
```

Move to the project

```bash
cd gcport
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

# Build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

Deploy to GitHub Pages

```bash
npm run deploy
```
